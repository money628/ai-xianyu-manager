# 数据库设计

## 数据库：SQLite

文件位置：`data/ai_storekeeper.db`

## 表结构（9 张表）

### 1. products — 商品快照

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| platform | TEXT | '1688' / 'pdd' / 'xianyu' |
| product_id | TEXT | 平台商品 ID |
| title | TEXT | 商品标题 |
| price | REAL | 价格 |
| sales_count | INTEGER | 销量 |
| seller_name | TEXT | 卖家名称 |
| seller_credit | TEXT | 卖家信用 |
| product_url | TEXT | 商品链接 |
| image_url | TEXT | 图片链接 |
| region | TEXT | 发货地 |
| raw_data | TEXT | 原始 JSON 数据 |
| fetched_at | TIMESTAMP | 抓取时间 |

唯一约束：`(platform, product_id, fetched_at)`

### 2. price_history — 价格曲线

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| platform | TEXT | 平台 |
| product_id | TEXT | 商品 ID |
| title | TEXT | 标题 |
| price | REAL | 价格 |
| sales_count | INTEGER | 销量 |
| snapshot_date | TEXT | 快照日期 (YYYY-MM-DD) |

唯一约束：`(platform, product_id, snapshot_date)`

### 3. opportunities — 套利机会

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| source_platform | TEXT | 进货平台 |
| source_title | TEXT | 进货商品标题 |
| source_price | REAL | 进货价 |
| source_url | TEXT | 进货链接 |
| target_platform | TEXT | 销售平台 |
| target_title | TEXT | 销售商品标题 |
| target_price | REAL | 销售价 |
| target_url | TEXT | 销售链接 |
| gross_profit | REAL | 毛利润 |
| net_profit | REAL | 净利润 |
| roi | REAL | 投资回报率 |
| confidence | REAL | 置信度 (0-1) |
| sales_signal | INTEGER | 销量信号 |
| status | TEXT | PENDING / APPROVED / REJECTED / ANALYZING / DECIDED |
| notes | TEXT | 备注 |
| discovered_at | TIMESTAMP | 发现时间 |
| decided_at | TIMESTAMP | 决策时间 |
| decided_by | TEXT | 决策人 |

### 4. user_blacklist — 黑名单

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| category | TEXT | 品类 |
| brand | TEXT | 品牌 |
| seller_name | TEXT | 卖家名 |
| product_keyword | TEXT | 商品关键词 |
| reason | TEXT | 原因 |
| created_at | TIMESTAMP | 创建时间 |

### 5. system_config — 系统配置

| 字段 | 类型 | 说明 |
|------|------|------|
| key | TEXT PK | 配置键 |
| value | TEXT | 配置值 |
| updated_at | TIMESTAMP | 更新时间 |

### 6. workflow_runs — 扫描日志

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| run_type | TEXT | 'auto_scan' / 'manual' / 'push' |
| platforms | TEXT | 涉及平台 |
| signals_found | INTEGER | 信号数 |
| opportunities_found | INTEGER | 机会数 |
| opportunities_pushed | INTEGER | 推送数 |
| status | TEXT | running / success / failed |
| error_message | TEXT | 错误信息 |
| started_at | TIMESTAMP | 开始时间 |
| finished_at | TIMESTAMP | 结束时间 |

### 7. push_log — 推送记录

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| opportunity_id | INTEGER | 关联机会 ID |
| push_type | TEXT | 'realtime' / 'daily_report' / 'test' |
| channel | TEXT | 'serverchan' / 'email' |
| status | TEXT | 'success' / 'failed' / 'skipped' |
| message | TEXT | 消息内容 |
| pushed_at | TIMESTAMP | 推送时间 |

### 8. keyword_pool — 关键词池

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| keyword | TEXT UNIQUE | 关键词 |
| source | TEXT | 'seed' / 'suggest' / 'manual' |
| seed_category | TEXT | 来源种子品类 |
| last_scanned_at | TIMESTAMP | 上次扫描时间 |
| scan_count | INTEGER | 累计扫描次数 |
| created_at | TIMESTAMP | 创建时间 |

## 索引

- products: (platform), (title), (fetched_at)
- price_history: (platform, product_id), (snapshot_date)
- opportunities: (status), (roi), (discovered_at)
- user_blacklist: (product_keyword), (brand), (seller_name)
- workflow_runs: (started_at)
- push_log: (opportunity_id), (pushed_at)
- keyword_pool: (last_scanned_at), (keyword)
