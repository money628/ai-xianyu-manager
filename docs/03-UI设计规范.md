# UI 设计规范

### 字体

| 用途 | 字体 | 替代方案 |
|------|------|----------|
| 正文 | Inter (400-700) | system-ui, sans-serif |
| 图标 | Material Symbols Outlined | — |
| 数字 | Geist / SF Mono | monospace |

**字体加载方式**：

- **禁止**：`<link href="https://fonts.googleapis.com/..." rel="stylesheet">` — Streamlit 可能被 CSP 拦截
- **必须**：`@import url('https://fonts.googleapis.com/...')` 写在 `<style>` 标签内
- **备选**：国内 CDN 镜像（如 `cdn.bootcdn.net`）作为降级

### 组件规范

#### 卡片 (glass-card)

```css
background: rgba(15, 19, 28, 0.75);
backdrop-filter: blur(12px);
border: 1px solid rgba(255,255,255,0.06);
border-radius: 12px;
padding: 0.75rem 1rem;
```

#### 徽章 (badge)

| 类型 | 背景 | 文字色 |
|------|------|--------|
| badge-danger (HIGH ROI) | `rgba(255,75,75,0.15)` | `#ff6b6b` |
| badge-warning (MID ROI) | `rgba(251,212,0,0.15)` | `#fbd400` |
| badge-default (LOW) | `rgba(180,180,180,0.12)` | `#a0a0a0` |

#### ROI 颜色

| 区间 | 颜色 |
|------|------|
| >= 60% | `#00f59b` (roi-high) |
| 30-60% | `#fbd400` (roi-mid) |
| < 30% | `#ff6b6b` (roi-low) |

### Streamlit 原生组件覆盖

- `st.metric` → 背景改为 `--surface-container`，hover 时边框变亮
- `st.button` → primary 按钮用 `--primary-container` 霓虹绿 + 发光阴影
- `st.progress` → 渐变色 `--primary-container` → `--secondary-container`
- `st.dataframe` → 暗色表格，大写表头
- `st.tabs` → 去掉原生下划线，card-style tab

### 五页布局

| 页面 | 路由 key | 核心组件 |
|------|----------|----------|
| 首页 | home | KPI 卡片 ×4、ROI 分布柱状图、Top5 玻璃卡片 |
| 雷达 | radar | 关键词池状态、Plotly 雷达图/热力图、自动扫描按钮、机会列表 |
| 看板 | kanban | 三列看板（待审/通过/拒绝）、黑名单表单 |
| 工具 | tools | 4 个 Tab（登录/参数/监控/黑名单）、利润预览计算器 |
| 历史 | history | 2 个 Tab（机会记录表/日报归档）、CSV/JSON 导出、30 日汇总 |
