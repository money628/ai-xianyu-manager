# 项目需求文档

## 项目背景

"AI店长 v1.1" 是一套个人本地工具，用于自动发现跨平台（1688/拼多多/闲鱼）价格套利机会。用户追求低成本、高效率，预算为零，所有功能基于免费抓取。

## 核心需求

### P0（必须实现）

1. **跨平台价差发现**：从 1688、拼多多获取货源，与闲鱼售价比较，找出利润空间
2. **自动品类发现**：种子品类（30 个） → Taobao Suggest API 自动扩展 400+ 子关键词
3. **净利润计算**：`净利润 = 售价 × (1 - 1.6%) - 进价 - 3元运费`
4. **ROI 实时推送**：ROI > 30% 自动推送通知（QQ 邮件，24h 去重）
5. **每日两次日报**：9:00 和 21:00 自动生成 Top10 日报
6. **看板审核**：人工审核每条机会，通过/拒绝决策

### P1（重要功能）

7. **黑名单管理**：屏蔽特定卖家、品牌、关键词
8. **抓取失败降级**：连续失败 ≥3 次自动切换为仅日报模式
9. **30 天数据保留**：自动清理过期数据
10. **价格曲线**：10 天历史价格追踪

### P2（锦上添花）

11. 导出 CSV/JSON
12. 平台登录态持久化
13. ROI 分布图表

## 约束条件

| 约束 | 详情 |
|------|------|
| 预算 | ¥0 |
| 抓取方式 | 免费 API + Playwright 无头浏览器 |
| 平台费率 | 闲鱼 1.6%，国内运费 ¥3/件 |
| 偏远地区 | 排除新疆/西藏/内蒙古偏远 |
| 推送方式 | QQ 邮箱 SMTP（465/SSL） |
| 广告成本 | 纯自然流量，¥0 |

## 非功能需求

- 单机运行，无需服务器
- SQLite 本地数据库
- Streamlit Web UI（Material Design 3 暗色主题）
- Windows 环境下运行
- 双击启动（bat 脚本）
