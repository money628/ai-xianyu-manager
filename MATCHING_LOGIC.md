# AI店长 匹配逻辑文档 — 供讨论优化

## 整体流程

```
关键词 → 闲鱼搜索 → 提取品牌关键词 → PDD搜索同类 → 标题相似度 → 校验 → ROI计算
```

## 1. 关键词提取 (extract_search_keywords)

从闲鱼标题中提取搜索词，用于在PDD搜索同类商品。

**当前逻辑:**
- 提取英文品牌/型号词（如 iphone16pro, casetify），合并版本号（iphone+16+pro+max → iphone16promax）
- 提取中文品类词（结尾为"壳/套/膜/线"等的2-gram）
- 组合为 [型号]+[品类] 查询

**示例:**
```
输入: "闪魔适用红米K80手机膜钢化膜防窥高清"
输出: ["k80 手机", "闪魔 手机"]
```

**问题:** 品牌词"闪魔"放在查询里，PDD可能搜不到同名品牌但搜索了不同品牌的同类品。

---

## 2. 标题相似度 (title_similarity)

计算闲鱼标题和PDD标题的相似度。

**当前逻辑:**
- 中文用2-gram切词（"手机膜" → "手机"+"机膜"）
- 英文用连续字母数字提取
- 过滤停用词（包邮/现货/新款/爆款 等）
- 60% TF-IDF余弦 + 40% Jaccard重叠

**示例:**
```
闲鱼: "闪魔适用红米K80手机膜钢化膜防窥高清"
PDD:  "闪魔适用红米k80手机膜红米k90max手机膜k70全覆盖"
相似度: 0.229
```

**问题:**
- 2-gram切词太粗糙，无法识别品牌实体
- 停用词列表缺少电商常见的促销词（"全新""未拆""包邮"等）
- 同品牌不同型号（K80 vs K90）相似度也较高，但实际不是同一个产品

---

## 3. 品牌匹配 (_extract_brand_keywords) ← 新增

**当前逻辑:**
- 提取英文单词（3字母以上）
- 提取已知中文品牌名（绿联/品胜/倍思/闪魔/图拉斯等，手动维护列表）
- 提取型号词（字母+数字，如 iphone16pro, k80）
- 要求闲鱼和PDD的品牌词集合有交集

**问题:**
- 中文品牌列表需要手动维护，新品牌/小众品牌无法识别
- "苹果"在两边都出现但实际上指手机型号而非品牌厂商
- 同品牌不同型号（K80 vs K90）会被认为品牌匹配

---

## 4. 价格比校验

**当前逻辑:**
- 价格比 = 闲鱼价 / PDD价
- 必须在 1.5~15 之间（严格版）
- 相似度 >= 0.30 时放宽价格比上限

**示例:**
```
闲鱼 ¥99 vs PDD ¥12.8 → 比率 7.7x → 通过
闲鱼 ¥9.9 vs PDD ¥9.8 → 比率 1.0x → 拒绝（价差太小）
```

## 5. 当前匹配效果

| 指标 | 宽松版（之前） | 严格版（现在） |
|------|---------------|----------------|
| 单关键词匹配数 | 25 | 1 |
| 真匹配 | 约1-2个 | 1个 |
| 假匹配 | 约23个 | 0 |
| 漏匹配 | 未知 | 可能有 |

**唯一的真匹配:**
```

ROI:  517%
```

## 6. 核心痛点

### 痛点A: 品牌实体识别弱
中文品牌只能靠硬编码列表识别，"闪魔"在列表里所以匹配到了。但"绿光魔盒"、"ROCK"、"TORRAS"等品牌未在列表，可能误匹配或漏匹配。

### 痛点B: 型号匹配不精确
"K80"和"K90max"是不同的产品，但当前只做字符串匹配不做语义区分。

### 痛点C: 相似度计算太粗糙
2-gram切词对中文电商标题效果差。"闪魔钢化膜K80防窥" 和 "绿光魔盒钢化膜iPhone16防窥" 的相似度可能高达0.2，因为"钢化膜""防窥"这些品类词产生了大量重叠。

### 痛点D: 闲鱼价格提取不准
Xianyu的价格分散在多行DOM中，经常提取错误（¥28.80可能被提取成¥28、¥6或¥19956）。

## 7. 可供讨论的优化方向

### 方向1: AI辅助匹配
- 用ChatGPT API比较两个标题是否是"同一商品"
- 返回匹配置信度 + 理由
- 成本: ~$0.001/次，每天扫描100个关键词约$0.5
- 优点: 准确性最高
- 缺点: 有成本、有延迟

### 方向2: 关键词分级权重
- 品牌词权重最高（闪魔>钢化膜）
- 型号词次之（K80>手机膜）
- 品类词最低
- 用加权Jaccard替代平权Jaccard

### 方向3: 结构化属性提取
- 从标题中解析: 品牌、型号、材质、功能
- 比较结构化属性而非纯文本
- 例如: {品牌:"闪魔", 型号:"K80", 品类:"钢化膜", 功能:"防窥"}

### 方向4: 商品ID/URL匹配
- 有些卖家在闲鱼和PDD用相同商品ID或SKU
- 从URL或商品详情中提取唯一标识
- 但大部分卖家不会这样做

### 方向5: 图搜匹配
- 下载闲鱼商品首图，用PDD图搜API找同款
- 或计算图片哈希相似度
- 优点: 视觉匹配比文本匹配更可靠
- 缺点: 需要下载和存储图片
