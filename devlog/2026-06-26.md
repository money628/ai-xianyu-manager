## 修复：KPI 卡片 HTML 渲染问题

### 问题
KPI 指标卡片的值区域显示原始 HTML `<span class="ui-ico...` 而非数字。原因是 `st.markdown` + `unsafe_allow_html=True` 在 Streamlit 1.58.0 中无法正确渲染嵌套的 `<div>` 内容。

### 解决方案
将 `metric_card` 函数从 `st.markdown` + HTML 改为使用 Streamlit 原生 `st.metric` 组件，确保兼容性。

### 验证
- Playwright 测试：7 个 stMetric 组件全部正确渲染，0 个 `span.ui-icon` HTML 标签
- 所有值显示正确数字，无 HTML 残留
